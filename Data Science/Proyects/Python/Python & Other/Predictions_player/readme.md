# Predicción de Desempeño de Jugadores de la NBA

Este proyecto permite predecir el desempeño de jugadores de la NBA utilizando diversos factores estadísticos. A través de un enfoque de *web scraping*, se obtiene información relevante sobre los jugadores y su rendimiento. El sistema procesa estos datos para generar predicciones sobre el futuro desempeño de los jugadores.

## Características
- Predicción de desempeño de jugadores de la NBA.
- Análisis de estadísticas relevantes como puntos, asistencias, rebotes, entre otras.
- Uso de *web scraping* para recolectar datos actualizados sobre los jugadores.

## Tecnologías
- Python
- Librerías de *web scraping* (por ejemplo, BeautifulSoup, Requests)
- Bibliotecas para análisis de datos (como Pandas y Numpy)
